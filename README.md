# deepseek-usage-monitor

Windows-first desktop tray monitor for DeepSeek API usage.

## Start

Double-click `deepseek-usage-monitor` in this folder, or use the `deepseek-usage-monitor` shortcut on the Windows desktop.

For development, the npm start command is still available:

```bat
npm run start
```

That builds the TypeScript files and starts Electron from a terminal.

## Folder Guide

- `src/`: application source code.
- `tests/`: Vitest test files.
- `scripts/`: helper scripts used by npm commands.
- `assets/`: app icon and visual assets.
- `dist/`: generated build output.
- `node_modules/`: installed npm dependencies.
- `package.json`: project metadata and commands.
